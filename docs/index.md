Distributaur is a super simple way to distribute rendering tasks across multiple machines.

## Welcome

This documentation is intended to help you understand the structure of the Distributaur API and codebase and how to use it to distribute rendering tasks across multiple machines for your own projects.

## Core Use Cases
Use Distributaur to distribute rendering tasks across multiple machines for the following use cases:

- Rendering videos
- Running simulations
- Processing data
- Any other task that can be parallelized

Please visit the [Getting Started](/getting_started) page to learn how to set up your environment and generate synthetic data.
